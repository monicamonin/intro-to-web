For This Task

This brief aims to be a basic introduction to the web, how the web works and how text and image work online. We will also introduce you to your own web space that you will be able to use throughout your Visual Communication degree. Sometimes you might use this for course work. But you are also free to use this as a play space!

You will design a simple web page that uses text from Emil Ruder's essay 'Rhythm'. We want to you to consider the typographic detailing (font, colour, size, spacing). the visual hierarchy and the combination of text and image.

As well the detailing the typography, we want you to create 2 images that will sit amoungst the text. These 2 images should be simple visual illustrations of a ideas about typography that the text discusses. For example one of your images could visualise the tension of narrow interword spacing and the other the visual effect of uniform spacing. Or you might use layout thumbnail sketches to show the rhythmic qualities that can be created with blocks of type. It is up to you what you want to draw out of the text and illustrate, but keep it very simple!*

Requirements / How to do This Task

We will be giving you a base file to work on that includes the 'Rhythm' essay. Stick to the base code, and using what we teach you in class we want you design the details of the page

  • Use CSS (Cascading Style Sheets) to alter the typography of the layout. You should seek to create a visual hierarchy in the typography (i.e. using the typographic detailing and white space)
  • Design two images that illustrate or visualise two ideas in the text. Include these images in your web page.
  • You can use the generic web fonts (i.e. ariel, times, verdana). But you should also aim to use web fonts. As they are free and accessible we will be using Google Web Fonts in class and recommend you stick to them for your assignment (for now!)

Working on Your Design

You can work directly with the code in order to achieve your design. Sometimes however it can be quicker to sketch out on paper what you are imagining it will look like. You can use thumbnail sketches to quickly detail the layout and visual hierarchy of your page.
